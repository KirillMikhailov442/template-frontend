# Панель управления устойствами от Ujin

Данный репозиторий является клиентской частью  приложения для управления устройствами ЖК.

## Команда
Десериализаторы

## Структура
Веб приложение соблюдает FSD (features slices design) архитектуру.
В архитектуру входят следующие директории:
* shared - (ui, lib, hooks, constants, config) - общедоступные, часто используемые компоненты, хуки, хелперы, константы и тд
* entities - (user, tempalte, device) - сущность со свойей бизнес логикой и уникальным функционал для конкретной сущности
* features - фичи использования (в проекте не использована)
* widgets - готовые блоки для использования на страницах
* pages - страницы приложения
* app - главные настройки (роутинг, глобальные стили, входной файл App.tsx)

## Запуск проекта
Нужно установить Node версии не ниже 21. Нужен пакетный менеджер NPM.

Создайте внутри проекта на уровне файла .env.example. Нужно создать локальный env файл по типу .env.lcoal или .env.development. В Новом env файле нужно указать путь к API приложения.
Добавльте в VITE_BACKEND_API_V1 значение http://194.87.102.27/api .

```bash
git clone https://github.com/KirillMikhailov442/template-frontend
cd template-frontend
git checkout dev
npm install
npm run dev
```

откройте браузер и введите в поиск: http://localhost:5173
